## 内容方向

- 🤖 AI 与大模型：如《AI 产品经理技术指南》《prompt 书写技巧》
- 🛠️ 开发实践与学习笔记

## 本地预览

如果你想在本地查看，直接用任意静态服务器打开根目录即可：

```bash
# 任选其一
python3 -m http.server 4000
# 或
npx serve .
```

然后浏览器访问 `http://localhost:4000`。
